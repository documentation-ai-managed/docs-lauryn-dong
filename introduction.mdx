## Overview

LAURYN DONG is a minimalist note-taking SaaS platform that lets you capture thoughts instantly, just like Flomo. You focus on ideas without distractions. Access it via web at `https://app.lauryndong.com` or mobile apps for iOS and Android. It syncs notes across devices in real-time.

Use LAURYN DONG for daily journaling, idea logging, or quick memos. Send notes via web form, API, email, or webhook. Search with full-text indexing for instant results.

<Callout kind="info">
  Sign up free at `https://app.lauryndong.com/register` to start capturing ideas in seconds.
</Callout>

  <Step title="Capture First Note" icon="edit-3">

    <CodeGroup tabs="JavaScript,cURL">
      ```javascript
      const response = await fetch('https://api.lauryndong.com/notes', {
        method: 'POST',
        headers: { 'Authorization': 'Bearer YOUR_API_KEY' },
        body: JSON.stringify({ content: 'My first note!' })
      });
      ```
      ```bash
      curl -X POST https://api.lauryndong.com/notes \
        -H "Authorization: Bearer YOUR_API_KEY" \
        -d '{"content": "My first note!"}'
      ```
    </CodeGroup>
  </Step>
