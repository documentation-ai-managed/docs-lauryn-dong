## Migration Guide

For major releases like v1.0.0, follow these steps to update smoothly.

<Steps>
  <Step title="Backup Data" icon="database">

    Export your journals:

````bash
npx lauryn-dong export --format=json --output=backup.json
````

  </Step>
  <Step title="Update Package" icon="download">

    <CodeGroup tabs="npm,yarn,pnpm">
````bash
npm install lauryn-dong@latest
````
````bash
yarn add lauryn-dong@latest
````
````bash
pnpm add lauryn-dong@latest
````
    </CodeGroup>

  </Step>
  <Step title="Run Migration" icon="settings">

    Apply schema changes:

````bash
npx lauryn-dong migrate --version=1.0.0
````

  </Step>
  <Step title="Verify" icon="check-circle">

    Test sync:

````bash
npx lauryn-dong test-sync --api-key=YOUR_API_KEY
````

  </Step>
</Steps>

<Callout kind="tip">
  If you encounter API errors, regenerate your key at `https://dashboard.example.com/settings`.
</Callout>
